# Collection Manager

Application auto-hébergée de gestion de collections basée sur des plugins déclaratifs.

## Images Docker

Des images prébuildées peuvent être publiées sur GitHub Container Registry :

- `ghcr.io/<owner>/collectionmgnt-backend`
- `ghcr.io/<owner>/collectionmgnt-frontend`

Voir [docs/deployment-docker.md](docs/deployment-docker.md) pour le déploiement local Docker/Podman et l'utilisation des images GHCR.

Architecture prête pour développement.
